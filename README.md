# YouTube Clone

## Project Overview
The YouTube Clone is a web application that mimics the user interface of the popular video streaming platform YouTube. It features a navigation sidebar, search functionality, video thumbnails, and interacts with a visually appealing layout utilizing Tailwind CSS. This project serves as an educational tool for those looking to learn about web development and responsive design.

## Installation
To run this project locally, follow the steps below:

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/youtube-clone.git
   ```
2. Navigate to the project directory:
   ```bash
   cd youtube-clone
   ```
3. Open `index.html` in your web browser. You can do this by double-clicking the file or using a local server setup.

No additional installation is required as all dependencies are included via CDN links in the HTML file.

## Usage
Once you have the project open in your browser, you can:

- Explore the sidebar links such as Home, Explore, Shorts, and Subscriptions.
- Use the search bar to type in queries (though functionality is not implemented).
- Browse through the displayed video thumbnails that represent video content.

## Features
- Responsive User Interface: The layout is designed to be fully responsive across different screen sizes.
- Search Bar: A search bar for user input (currently non-functional as no backend is implemented).
- Sidebar Navigation: Easily navigate through different sections of the application.
- Video Thumbnails: Display of video content with thumbnails, duration, title, and channel information.

## Dependencies
This project utilizes the following dependencies:
- Tailwind CSS: A utility-first CSS framework for creating designs directly in your markup.
- Font Awesome: A font and icon toolkit for including icons in the project.
